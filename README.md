# Dynamic DRS Quality Platform

## Overview

The Dynamic DRS Quality Platform is a web-based quality assessment system designed for materials, products, tools, equipment, laboratory samples, and manufacturing processes.

Unlike fixed-domain quality systems, the platform allows users to define custom parameters and build project-specific DRS models.

---

## Features

### Project Management

* Multiple independent projects
* CSV-based projects
* Manual parameter projects
* Project deletion and management

### Parameter Management

* Add custom parameters
* Edit parameter limits
* Remove parameters
* Engineering limit definitions

### DRS Engine

* Dynamic DRS generation
* Statistical model generation from historical datasets
* Continuous learning from accepted samples
* PASS / WARNING / REJECT prediction

### Sample Evaluation

* Real-time quality assessment
* Parameter-by-parameter analysis
* JSON export
* Historical sample storage

### Data Management

* SQLite database
* Historical sample tracking
* Accepted sample learning
* Rejected sample traceability
